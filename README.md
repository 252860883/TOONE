## 关于 TO ONE
>TO ONE 是一款独立游戏，以小数的二进制数作为剧情主线展开。玩家从“0”关开始，将面对几十道由不大于 1 的小数组成的递增关卡。在每道关卡中，由当前关卡小数的二进制数铺满游戏宫格，玩家需要通过移动色块将所有的 0 变成 1。最终达到“1”关时既在整个 TO ONE 游戏人生中通关。
